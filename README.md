# Movie List Application

A simple web application that allows users to add and remove movies from a list. The movie details, including images, are fetched from the OMDB API, and the list is persisted in local storage.

## Features

- Add movies to the list by entering the movie name.
- Remove movies from the list.
- Display movie titles along with their respective images.
- Persist the movie list in local storage, so it remains after page reloads.

## Technologies Used

- HTML
- CSS
- JavaScript
- OMDB API
